# 📊 Sales Data Analysis with Python

Projeto de análise de dados de vendas utilizando **Python e Pandas**.

## 🚀 Objetivo

Analisar um conjunto de dados de vendas para identificar padrões, faturamento e desempenho de produtos.

## 🛠 Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📂 Estrutura do projeto

dataset com dados de vendas
notebook com análise exploratória
gráficos gerados a partir dos dados

## 📈 Análises realizadas

* faturamento por produto
* vendas por região
* análise de desempenho de categorias

## 📊 Exemplo de visualização

Gráfico de faturamento por produto.
